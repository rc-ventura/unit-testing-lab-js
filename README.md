# 🧪 Testes Unitários em JavaScript

Repositório de laboratórios práticos de testes unitários com **Vitest**, cobrindo desde os fundamentos até recursos avançados como mocks, spies e testes assíncronos.

---

## 📁 Projetos

| # | Pasta | Tema | README |
|---|---|---|---|
| 1 | `01-unit-test-basic` | Testando código existente (E-commerce) | [📖 Ver aula](./01-unit-test-basic/README.md) |
| 2 | `02-calc-project` | TDD — Calculadora TechStore | [📖 Ver aula](./02-calc-project/README.md) |
| 3 | `03-tdd-project` | TDD — Sistema de Boletim Escolar | [📖 Ver aula](./03-tdd-project/README.md) |

Cada pasta contém **um projeto independente** com seu próprio `package.json` e instruções completas.

## � Como começar

```bash
# Entre na pasta do projeto desejado
cd 01-unit-test-basic   # ou 02, 03, 04

# Instale as dependências
npm install

# Rode os testes
npx vitest
```
